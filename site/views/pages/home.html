Esta herramienta tiene como objetivo real mostrar el funcionamiento de los componentes para desarrollo de aplicaciones TAC generados por tacteam.
Durante el desarrollo de la tienda TAC, se extrajeron algunos componentes para simplificar la creación de nuevas aplicaciones.
Este sitio puede ser navegado con eventos emitidos por un control remoto, sin necesidad de conectar un teclado o mouse. En este caso se emulan los eventos del control remoto a trave del teclado.
No obstante, es posible la integracion de la navegación por control remoto, teclado y mouse.
Para modificar campos de texto tambien se dispone de teclados virtuales.
